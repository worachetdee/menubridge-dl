Changelog

Track the evolution of the MenuBridge Design System.

v1.2 - Examples & Components

January 28, 2026

Added implementation examples and enhanced component flexibility.

  • Examples: Added Login Page and Create Menu implementations with preview mode.
  • Components: Updated <menu-card> to support custom badge icons.
  • Components: Updated <menu-button> to support full-width layout.
  • Fixes: Corrected sidebar link generation for subdirectories and updated copyright year.

v1.1 - Sidebar & Refinement

January 28, 2026

Major navigation refactor and documentation improvements.

  • Navigation: Switched from top header to a fixed sidebar layout for better scalability.
  • Typography: Added detailed Type Scale table and usage guidelines (Epilogue vs Inter).
  • Branding: Official rename to "MenuBridge".

v1.0 - Initial Release

January 28, 2026

Initial release of the MenuBridge Design System foundation.

  • Foundations: Definition of Color Palette and Typography.
  • Components: Buttons, Badges, and Translation Cards.
  • Architecture: Lightweight, zero-build HTML/JS structure with Tailwind CSS.
