code Component Usage

  • QR Code: Inline SVG placeholder for the QR code, wrapped in a gray background container.
  • Share Link: Read-only input with a copy button using the content_copy Material icon.
  • Card Layout: Two-column grid for QR/share cards, full-width card for menu details.
  • Buttons: Primary dark button for main action, outline button for secondary actions.
