  • Accordion Items: Native <details> elements for expandable menu items with CSS-only chevron rotation.
  • Dietary Icons: Color-coded circles (emerald = vegetarian, blue = fish/shellfish, red = meat) with Material icons.
  • Guest Notes: Amber-tinted callout box for restaurant-wide notices and policies.
  • Section Tags: Blue badge labels to categorize menu sections (alacarte, set menu, etc.).
