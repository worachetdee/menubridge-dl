schema Anatomy

1

Top Bar

App logo, language toggle (EN/JP), and user avatar — persistent across all authenticated views.

2

Onboarding Card

Dismissible info card explaining core features — shown to new users, hidden via close button.

3

Shared Menu

Displays the currently published menu with image preview, QR code, metadata, and action buttons (view, edit, copy link, download QR).

4

Replace Menu

CTA area to replace the current published menu — routes to the Create Menu flow.

5

Previous Menus

Version history list showing past menu iterations with timestamps and view links.
