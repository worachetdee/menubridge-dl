code Implementation Notes

  • Sections: The form is broken into logical sections using bg-white rounded-xl border border-[#eaf1f1] cards to reduce cognitive load.
  • Form Grouping: Labels (text-sm font-bold) are clearly separated from help text (text-xs text-[#5c8a8a]).
  • Upload State: The dashed border area uses standard Tailwind properties border-2 border-dashed and hover states for interactivity.
  • Components: Reuses <menu-button> for consistent calls to action.
