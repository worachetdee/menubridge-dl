schema Anatomy

1

Course Section

Omakase course with expandable course items. Each item has inline edit (name, ingredients, preparations) and dietary icons.

2

Guest Notes

Editable notice text shown to guests, with bilingual content about the restaurant's serving style.

3

À la Carte Sections

Multiple sections (おつまみ, 主菜, 前菜, 前方物, 〆) each containing menu items with prices, edit/delete controls.

4

Inline Item Row

Each menu item shows Japanese name, price, and edit/delete actions. Clicking Edit opens the detailed edit interaction (see Edit Menu Item page).

5

Add Controls

Add Course Item, Add Menu Item per section, and Add Section at the bottom — all with bilingual labels.

6

Preview CTA

Sticky footer button to preview the shared menu layout before publishing, styled with the tertiary (terracotta) color.
