code Component Usage

  • Layout: Grouped form sections use bordered rounded-xl cards to visually separate concerns (profile, language, security).
  • Inputs: Standard text inputs with border-[#eaf1f1] borders. The password field is disabled with a separate "Change Password" action button.
  • Danger Zone: The Delete Account section uses bg-red-50 with border-red-200 to signal a destructive action. Button is disabled to indicate the feature is unavailable.
  • Actions: Sticky footer with Cancel (text button) and Save (primary <menu-button>) aligned to the right.
