schema Anatomy

1

Item Name & Price

Editable fields for the Japanese item name and price. Editing the name triggers a regeneration warning.

2

Reorder & Done Controls

Up/down arrow buttons to reorder within the section, and a Done button to confirm changes.

3

Delete Item

Centered destructive action button — removes the menu item from the section.

4

Guest Explanation Components

AI-generated explanation cards for each ingredient component. Each card has an Include/Hide toggle to control visibility in the guest-facing menu.

5

Dietary Information

Icon-based dietary tags (Vegetarian, Fish/Shellfish, Meat). Active/suggested tags show a darker border. AI suggests tags based on ingredients.
