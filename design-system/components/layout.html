Library

Layout

A zero-build utility for injecting shared navigation and managing consistent page structure.

Overview

The MenuBridge Design System uses a lightweight, client-side approach to layout management. Instead of relying on a complex build step or server-side rendering for the shell, we use a simple JavaScript utility (layout.js) to inject the sidebar and handle active states.

  • Zero Dependencies: No framework required, just vanilla JS.
  • Automatic Active States: Detects current URL to highlight the correct sidebar link.
  • Path Resolution: Automatically handles relative paths (./ vs ../) for nested pages.

Usage

1. Include the Script

Add the script tag to your HTML head. Ensure it runs after the DOM is ready (handled internally by the script).

<script src="../layout.js"></script>

2. Add the Sidebar Container

Place the empty aside element where you want the sidebar to appear.

<aside id="app-sidebar"></aside>

3. Structure Your Content

Wrap your main content in a main tag. The layout script will automatically add the necessary left margin on desktop.

<main class="flex flex-1 flex-col items-center pb-20 mt-16 lg:mt-0">
  <!-- Page Content -->
</main>
