Library

Layout

A zero-build utility for injecting shared navigation and managing consistent page structure across all screen sizes.

Overview

The MenuBridge Design System uses a lightweight, client-side approach to layout management. A single JavaScript utility (layout.js) injects the sidebar navigation and handles active states — no framework or build step required.

  • Zero Dependencies: Vanilla JS, no framework required.
  • Automatic Active States: Detects current URL via path.endsWith() to highlight the correct sidebar link.
  • Path Resolution: Automatically handles relative paths (./ vs ../) based on subfolder depth.
  • Responsive: Fixed sidebar on desktop (lg+), collapsed horizontal nav on mobile.

Page Structure

Every page in the design system follows this consistent shell structure. The sidebar is injected by layout.js at runtime.

<!-- Page Shell -->
<body class="bg-background-light text-[#101818] antialiased">
  <div class="relative flex min-h-screen w-full flex-col lg:flex-row overflow-x-hidden">

    <!-- Sidebar (injected by layout.js) -->
    <aside id="app-sidebar"></aside>

    <!-- Main Content (layout.js adds lg:ml-64) -->
    <main class="flex flex-1 flex-col items-center pb-20 mt-16 lg:mt-0">
      <section class="w-full max-w-[1200px] px-6 py-12 lg:px-40">

        <!-- Page Header -->
        <div class="mb-12">
          <p class="text-primary font-bold uppercase text-xs mb-2">Category</p>
          <h1 class="font-display text-4xl md:text-5xl font-black">Title</h1>
          <p class="text-[#5c8a8a] text-lg mt-4 max-w-2xl">Description</p>
        </div>

        <!-- Content Sections... -->

      </section>
    </main>
  </div>
</body>

Setup

1. Include Scripts

Add the config and layout scripts to your HTML head. Both run after DOMContentLoaded.

<script src="../config.js"></script>
<script src="../layout.js"></script>

2. Add the Sidebar Container

Place the empty aside element as the first child inside the flex wrapper.

<aside id="app-sidebar"></aside>

3. Structure Your Content

Wrap page content in main. The layout script automatically adds lg:ml-64 and w-full at runtime.

<main class="flex flex-1 flex-col items-center pb-20 mt-16 lg:mt-0">
  <section class="w-full max-w-[1200px] px-6 py-12 lg:px-40">
    <!-- Your content -->
  </section>
</main>

Active State Logic

Sidebar links use path.endsWith('/filename.html') for exact matching. The leading / prevents false positives (e.g., preview-menu.html matching review-menu.html).
