Usage Guidelines

check_circle

Do

  • Use the spacing scale consistently (multiples of 4px)
  • Increase spacing to separate unrelated content groups
  • Use gap for flex/grid children instead of margins
  • Match padding to element importance (more space = more prominence)
error

Don't

  • Use arbitrary pixel values outside the scale
  • Mix margin and padding for the same spacing purpose
  • Collapse spacing between related elements to zero
  • Use large spacing values (16+) for inline element gaps
