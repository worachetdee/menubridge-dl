Usage Guidelines

check_circle

Do

  • Use Outlined style consistently throughout the app
  • Pair icons with text labels for clarity
  • Match icon size to adjacent text size
  • Use semantic color to reinforce icon meaning
error

Don't

  • Mix Outlined and Filled styles in the same context
  • Use icons without accessible labels or aria attributes
  • Resize icons with width/height instead of font-size
  • Use decorative icons that add no meaning

Standard Pattern

<span class="material-symbols-outlined">icon_name</span>
