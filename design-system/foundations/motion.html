Usage Guidelines

check_circle

Do

  • Use transitions for interactive feedback (hover, focus, active)
  • Keep durations under 300ms for micro-interactions
  • Use ease-out for enter animations, ease-in for exit
  • Animate opacity and transform (GPU-accelerated)
error

Don't

  • Animate layout properties (width, height, top, left)
  • Use durations longer than 500ms for UI transitions
  • Add animation to non-interactive elements
  • Ignore prefers-reduced-motion
