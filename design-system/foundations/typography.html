auto_stories When to use Epilogue

  • Marketing headers and feature titles
  • Storytelling elements (hero sections, quotes)
  • High-impact numbers or data points
  • Do not use for long-form paragraphs or UI labels

wysiwyg When to use Inter

  • Standard paragraph text (descriptions, articles)
  • User Interface elements (buttons, inputs, tooltips)
  • Dense information displays (tables, lists)
  • Do not use for primary brand headings
